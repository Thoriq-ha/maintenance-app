# Maintenance App
Railway equipment maintenance application using a QR code scanner

## Installation

#### Step 1
Download or clone this repo by using the link below:

```bash
https://github.com/Thoriq-ha/maintenance-app
```

#### Step 2
Download or clone this repo by using the link below:

```bash
flutter pub get 
```


#### Step 3
Download or clone this repo by using the link below:

```bash
flutter pub run 
```
